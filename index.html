OTRSADConfig

OTRS is a flexible web-based ticketing system used for Customer Service & Help Desk. It's fast to implement and easy to customize.

This application allows server administrators to generate the configuration needed for OTRS to authenticate its customers against a Microsoft Active Directory server, using a graphical interface. Includes functions to retrieve the required information automatically, in order to simplify the process. In addition, all required fields include helpful hints to guide in the configuration process.
